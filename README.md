# Assignment #03: Data Analysis & Visualization using FIFA 21 Dataset

## Objective
This assignment demonstrates the complete data-science workflow using Python, including:

- NumPy fundamentals
- Pandas data manipulation
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Visualizations using Matplotlib & Seaborn

## Dataset
- Dataset imported directly from Kaggle using the Kaggle API in the notebook.
- Dataset description: FIFA 21 Complete Player Dataset containing player attributes such as:
  - Name, Age, Nationality, Club_name
  - Overall rating, Value, Wage, potential
  - Skill stats like Shooting, Passing, Dribbling, Defending, Physical
- Source: [Kaggle FIFA 21 Dataset](https://www.kaggle.com/datasets/stefanoleone992/fifa-21-complete-player-dataset/data)

## Contents of Repository
1. `Assignment03.ipynb` - Complete analysis notebook including:
   - Part 1: NumPy operations (array properties, vectorized operations, indexing, slicing, broadcasting)
   - Part 2: Pandas operations (data cleaning, manipulation, filtering, grouping, aggregation)
   - Part 3: Data visualization with Matplotlib (line plot, bar chart, histogram, scatter plot, box plot)
   - Part 4: Advanced visualization with Seaborn (heatmap, pairplot, violin plot)
